# Проверка СНИЛС (SQLite)

Реализация проверки контрольного числа СНИЛС через CHECK-ограничение.

## Файлы
- `Script-validation.sql` — создание таблицы с валидацией
- `Script-check.sql` - добавление данных в таблицу для проверки работоспособности кода
 
## Использование
Открыть в DBeaver, выполнить скрипт (Alt + X).
